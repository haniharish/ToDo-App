#  Todo List App (JavaScript)

A simple Todo List application built using **HTML, CSS, and JavaScript**.  
This project allows users to **add** and **delete** tasks, helping understand core JavaScript concepts and DOM manipulation.

---

##  Features
-  Add new tasks
-  Delete existing tasks
-  Display all tasks dynamically
-  Instant UI update without page reload

---

## Technologies Used
- HTML  
- CSS  
- JavaScript (Vanilla JS)
